# Data Science & ML Roadmap

Repositorio personal con mi recorrido completo desde fundamentos
hasta MLOps. Plan de ~8 meses estructurado en 7 fases con
proyectos de portafolio.

## Estructura
- 00_setup/ — Configuracion del entorno
- 01_python_for_data/ — Python profesional para datos

## Como replicar el entorno
git clone https://github.com/FelipeA-del/data-science-ml-roadmap.git
cd data-science-ml-roadmap
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
